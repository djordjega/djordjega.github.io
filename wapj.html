Web Apps Project - Java

Basic idea was to explore different tools for creating Java based web apps/services/apis. FrontEnd is made of one page powered by AngularJS. It sends GET, POST and DELETE requests and displays the response data. BackEnd processes the requests, works with MySQL database, and returns the response.

This is still an active project so there will be more soon. Probably: JSF, PLAY, GWT, GRAILS, VAADIN, NETTY...
